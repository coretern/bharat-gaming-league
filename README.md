# ArenaX Esports Platform

ArenaX is a premium, futuristic esports tournament platform for BGMI and Free Fire. Built with Next.js 16 and Tailwind CSS 4.

## Features
- 🔥 Hero section with gaming stats
- 🏆 Filterable tournament listings
- 📝 Team registration with payment verification
- 📊 User dashboard & statistics
- 🛠 Admin panel for event management

## Tech Stack
- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Notifications**: React Hot Toast

## Development
```bash
npm install
npm run dev
```
